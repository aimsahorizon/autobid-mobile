How It Works:
=============

When user clicks In Transaction card:
1. listings_grid.dart receives listing with id 'list_005'
2. Creates TransactionController with TransactionMockDataSource
3. Navigates to PreTransactionPage with transactionId='list_005'
4. PreTransactionPage calls controller.loadTransaction('list_005')
5. TransactionMockDataSource.getTransaction('list_005'):
   - Tries to find existing transaction with id 'list_005'
   - Not found, calls _createDynamicTransaction('list_005')
   - Returns new TransactionEntity with all required fields
6. Similarly for chat messages and timeline
7. UI displays working transaction page with sample data

Same pattern for Draft and Cancelled tabs.

Backend Switching:
==================

2. Implement actual Supabase queries in TODO sections
